Les exemples sont répartis en deux dossiers :

- numeros : contient des grapes au format 1 (numéros de sommets)
- noms : contient des grapes au format 2 (noms de sommets)

Certains graphes ne sont pas disponibles dans les deux formats.

Voici le contenu et les sources des différents graphes :

- ex1*.txt :
  Graphe d'exemple du cours sous différentes formes admissibles :
    - avec sauts de lignes (lr),
    - avec tabulations (tabs),
    - avec arcs mélangés (scrambled)

- ex2.txt :
  Un autre graphe d'exemple

- automaton.txt :
  Un exemple d'automate discret
  Source : https://graphviz.org/Gallery/directed/fsm.html

- only-one-scc.txt :
  Un exemple de graphe contenant une seule CFC
  Source : https://graphviz.org/Gallery/neato/philo.html

- PC-*.txt :
  Différents graphes issus de la base de données biologique PathwayCommons, représentant des interactions gééntiques, protéiques et modléculaires : https://www.pathwaycommons.org/
  Source : https://download.baderlab.org/PathwayCommons/PC2/v12/
  Les différents modèles sont :
    - bind : https://en.wikipedia.org/wiki/Biomolecular_Object_Network_Databank#Biomolecular_Interaction_Network_Database_(BIND)
    - biogrid : https://wiki.thebiogrid.org/doku.php/aboutus
    - netpath : https://en.wikipedia.org/wiki/NetPath
    - psp : https://www.phosphosite.org/

- GD06-Java.txt :
  Le graphe des dépendances de la bibliothèque standard du langage Java.
  Source : https://sparse.tamu.edu/Pajek/GD06_Java
